# 🚀 ALGOLAB — Hybrid Merge + Insertion Sort Visualizer

A futuristic **ADA (Analysis and Design of Algorithms)** visualization platform built to demonstrate the working of a **Hybrid Merge + Insertion Sort Algorithm** through immersive animations, real-time analytics, and interactive algorithm comparison systems.

🔗 [Open ALGOLAB Visualizer](https://hybridsort-visualizer.vercel.app/)

---

## 🌌 Project Overview

ALGOLAB is an advanced algorithm visualization engine designed to make sorting algorithms more interactive, understandable, and visually engaging.

The project combines:

* ⚡ Merge Sort
* ⚡ Insertion Sort
* ⚡ Hybrid Optimization Logic

into a cinematic cyberpunk-style dashboard with:

* real-time visualization
* live analytics
* performance tracking
* intelligent recommendations
* algorithm comparison systems

This project was developed as an ADA micro-project to demonstrate practical optimization techniques used in modern algorithm engineering.

# 🧮 Algorithms Used

## 🔹 Merge Sort

Efficient divide-and-conquer sorting algorithm with:

* Time Complexity: **O(n log n)**
* Space Complexity: **O(n)**

---

## 🔹 Insertion Sort

Efficient for smaller or nearly sorted datasets.

* Best Case: **O(n)**
* Worst Case: **O(n²)**

---

## 🔹 Hybrid Sort

Combines Merge Sort and Insertion Sort to improve practical runtime efficiency by reducing recursive overhead for smaller partitions.
